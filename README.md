# alx-intermediate-frontend
## ALX Intermediate Frontend - Semantic HTML Project
Project Overview
This project focuses on building a solid foundation in Semantic HTML while emphasizing accessibility, SEO optimization, and the implementation of ARIA roles for enhanced usability. Through incremental tasks, learners will structure and enhance a web page to make it not only visually structured but also optimized for screen readers and search engines.

Learning Objectives
By completing this project, you will:

Master Semantic HTML structure using elements like <header>, <main>, <article>, <section>, and <footer>

Understand how to optimize web pages for SEO through proper meta tags and document structure

Implement ARIA roles and attributes to improve accessibility

Learn form design best practices for accessibility

Practice progressive enhancement through incremental development

Project Structure
The project consists of 4 incremental HTML files:

0-index.html - Basic semantic structure

1-index.html - Enhanced with meta tags for SEO

2-index.html - Complete blog post layout

3-index.html - Added accessible form with ARIA
